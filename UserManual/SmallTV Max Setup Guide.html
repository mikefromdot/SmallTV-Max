1
Power On & First Boot

Plug the USB-C cable into the device and a power source. The SmallTV Max boots in a few seconds and shows the boot screen.

✅
What you'll see
The orange SmallTV Max boot screen appears for a few seconds while the device initialises. If a GIF is already loaded on the SD card it will start playing automatically.
⚠️
SD Card Note
If you insert an SD card, it must be formatted as FAT32. If the card is exFAT (common on cards larger than 32 GB) the device will show a format error and continue without the SD card. Use your computer to reformat it as FAT32 before inserting.
2
Connect to SmallTV Max Wi-Fi

If no Wi-Fi credentials have been saved, the device enters Setup Mode and broadcasts its own Wi-Fi network. Scan the QR code on the screen or connect manually.

On your phone or computer, open Wi-Fi settings and connect to the network named SmallTV Max. No password is required.

📱 iPhone / Android

A browser may open automatically. If not, open Safari or Chrome and navigate to 192.168.4.1

💻 Windows / Mac

Open any browser after connecting and navigate to http://192.168.4.1 — the captive portal does not appear automatically on Windows by design.

📖
Read Me QR Code
Point your phone camera at the QR code shown at the bottom of the setup screen to open the full online user manual on GitHub.
3
Open the Setup Page

Navigate to http://192.168.4.1 in your browser. You'll see the SmallTV Max web interface with the Wi-Fi setup form.

4
Enter Your Wi-Fi Credentials

Find the Wi-Fi card in the web interface and enter your home network details.

Network name (SSID) — type the exact name of your home Wi-Fi network, including capital letters.
Password — enter your Wi-Fi password. Leave blank if your network has no password.
Hostname (optional) — a friendly name for the device on your network, e.g. smalltv-max. Defaults to smalltv-max if left empty.
Click Save & Connect — the device saves your credentials and reboots. It will connect to your network automatically from now on.
⚠️
Wi-Fi Disconnection
Your browser will lose connection when the device leaves Setup Mode. Wait about 15 seconds, reconnect your computer to your home Wi-Fi, then continue to Step 5.
5
Access the Web Interface

Once connected to your home network, the device is accessible via its IP address or hostname.

🔗 By hostname

Navigate to http://smalltv-max.local in your browser. Works on macOS, iOS, and most Android devices. May not work on all Windows setups.

🔢 By IP address

Find the device IP on your router's admin page, or check the Info screen on the device. Then navigate to e.g. http://192.168.1.42

💡
Bookmark it
Once you find the address, bookmark it in your browser for quick access in future.
6
Using the Web Interface

The web interface gives you full control over every aspect of the device.

📡 Wi-Fi

Change the connected network, password, or device hostname at any time.

🕐 Clock & Time Zone

Set your NTP server and POSIX timezone string so the clock shows the correct local time.

🌤 Weather

Enter a free OpenWeatherMap API key and your city. Choose metric or imperial units and enable or disable the weather display mode.

🖼 Display Mode

Switch between GIF player, Clock, Info, and Weather modes from any browser.

☀️ Brightness

Drag the brightness slider. The setting is saved automatically.

🎞 GIF Manager

Upload GIF files, set how long each one plays before advancing, and rescan the storage after changes.

⬆️ Firmware Updates

The device checks GitHub for updates automatically. The Update Available banner appears in the OTA section when a new version is ready — just click Install.

🔧 Device

Reboot, factory reset, or format the SD card from the Device section.

8
Adding GIF Files

The GIF player supports animated GIF files stored on an SD card (preferred) or internal storage (SPIFFS). Up to 32 GIF slots are supported.

💾 SD Card (recommended)

Format the card as FAT32. Copy your GIF files into a folder named /gifs on the card. Insert the card and reboot — the device scans automatically.

📤 Upload via web UI

Open the GIF Manager section, click Choose File, select a GIF, and click Upload. The device rescans storage automatically after each upload.

💡
Best results
GIFs display best at or near 320 × 480 pixels (portrait). Smaller GIFs are centred on a black background. There is no enforced file size limit but very large GIFs require more PSRAM — aim for under 10 MB per file for smooth playback.
⏱
Setting display duration
In the GIF Manager, each file has a duration slider (in seconds). Set it to 0 to loop the GIF indefinitely, or set a number of seconds before it advances to the next one.
9
Firmware Updates

SmallTV Max checks for firmware updates automatically and lets you install them from the web interface with one click.

Automatic check — 60 seconds after connecting to Wi-Fi, the device silently checks GitHub for a newer version. This repeats every 24 hours.
Update available banner — if a new version exists, an amber banner appears in the Firmware Update section of the web UI showing the version number.
Click Install — the device downloads the firmware directly from GitHub and flashes it. A full-screen progress overlay appears during flashing. Do not power off the device during this process.
Automatic reboot — once flashing is complete, the device reboots and the web page reloads automatically. All your settings are preserved.
192.168.1.42 — Firmware Update (OTA)
🔒
Settings are always preserved
Wi-Fi credentials, NTP settings, weather configuration, brightness, display mode, and GIF durations are stored in NVS (non-volatile storage) and survive firmware updates completely.

Troubleshooting

🌐 Can't reach 192.168.4.1

Make sure your device is connected to the SmallTV Max Wi-Fi network, not your home network. On Windows, navigate manually — the captive portal does not pop up automatically.

🔄 Device won't connect to my Wi-Fi

Double-check the SSID and password are exactly correct, including capitalisation. The device will fall back to Setup Mode and broadcast its network again if connection fails.

📁 SD card not recognised

The card must be formatted as FAT32. Cards larger than 32 GB are often formatted exFAT by default — use your computer's disk utility to reformat as FAT32 before inserting.

🌡 Weather not showing

Ensure you have a valid OpenWeatherMap API key (free tier works), a valid city name e.g. London,UK, and that weather is enabled in the web interface.

⏱ Clock shows wrong time

Check the NTP settings in the web interface. Set the correct POSIX timezone string for your region. The device requires an internet connection to sync time.

🔁 Factory Reset

In the Device section of the web interface, click Factory Reset. This clears all saved settings (Wi-Fi, NTP, weather, brightness) and reboots into Setup Mode. GIF files on the SD card are not affected.
